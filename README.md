# Traction Force Microscopy – Summer Project

This repository documents my summer research project on
fabrication and use of PDMS/hydrogel substrates for traction force microscopy (TFM).

## Project Goals
- Fabricate soft PDMS substrates for TFM
- Track bead displacements and traction forces
- Correlate time‑resolved traction forces with actin network dynamics
- Investigate cytoskeletal metastability

## Current Progress
- Learning PDMS substrate fabrication
- Reviewing TFM theory and assumptions

## Next Steps
- Optimize substrate thickness and stiffness
- Implement bead displacement tracking
